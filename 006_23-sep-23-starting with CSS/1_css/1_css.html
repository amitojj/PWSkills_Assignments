Question:

Create a simple page with some div tags and show different ways to add CSS as well as what happens when you target the same elements with inline, internal, and external CSS. Also, utilize comments in the project where required.

Answer:

There are three ways to add CSS. via external file . Via internal style tags in head part of the page and as inline CSS. Out of these three inline css has highest priority. As we can see in this document we targetted h2 in div class answer via internal css and inline css. but we can see inline css is applied. Internal css has priority over External css . In this document body element is targetted via both external and internal css, But internal css is applied.
